🌤️ Weather App — Техническое задание
📌 1. Цель проекта

Разработать веб-приложение для отображения текущей погоды и прогноза на несколько дней для любого города мира. Приложение должно быть быстрым, удобным и адаптивным, с использованием внешнего API погоды.

🎯 2. Основной функционал
🔎 2.1 Поиск города
Пользователь вводит название города
Автодополнение (опционально)
Обработка ошибок (город не найден)
🌡️ 2.2 Текущая погода

Отображается:

Температура (°C / °F)
Ощущается как (feels like)
Погодные условия (ясно, дождь, снег и т.д.)
Иконка погоды
Влажность (%)
Давление
Скорость ветра
Время обновления данных
📅 2.3 Прогноз погоды (5–7 дней)

Для каждого дня:

Минимальная и максимальная температура
Иконка состояния
Краткое описание
День недели
📍 2.4 Геолокация пользователя
Кнопка “Погода рядом”
Определение координат браузера
Автоматический вывод погоды
⭐ 2.5 Избранные города
Добавление города в избранное
Быстрое переключение между городами
Хранение в LocalStorage
⚙️ 3. API и данные
🌐 Используемое API:
OpenWeather API (или аналог)
📥 Данные:
Current Weather API
7-day Forecast API
Geocoding API (поиск города)
🧠 4. Логика приложения
Основные сценарии:
Пользователь открывает приложение
Вводит город
Получает:
текущую погоду
прогноз на 7 дней
Может сохранить город в избранное
Может переключаться между городами
🎨 5. UI / UX требования
Общий стиль:
Минимализм
Темная и светлая тема (желательно)
Большие цифры температуры
Иконки вместо текста где возможно
Экраны:
Главный экран (погода)
Поиск
Избранные города
📱 6. Адаптивность
Mobile First
Корректное отображение на:
телефонах
планшетах
десктопе
🧱 7. Технический стек (рекомендуемый)
Frontend:
React / Next.js
Tailwind CSS или CSS Modules

fetch / axios
Хранение:
LocalStorage (избранное)

🧾 11. Метрики готовности проекта

Проект считается завершённым если:

Работает поиск города
Показывается текущая погода
Есть прогноз на 5–7 дней
Есть избранные города
Есть адаптивность
